# Kingshot Event Tracker — User Documentation

Welcome! The Kingshot Event Tracker is a web app that helps kingdoms and alliances keep score of in‑game events, track their players, and see who earned rewards. This documentation is written for the people who *use* the app — no technical or coding knowledge is assumed.

If you are brand new, start with **[What the Tracker Does](getting-started/what-is-the-tracker.md)** and keep the **[Glossary](getting-started/glossary.md)** open in another tab. The glossary defines every term (template, instance, review row, grant, effective plan, and so on) that the rest of the docs rely on.

## Find your guide by role

Everyone in the app has a **role** that decides what they can see and do. If you are not sure which role you have, look at the top of any page — your role is shown in the context banner. See **[Roles Explained](roles/overview.md)** for the full picture.

| Your role | Start with |
|---|---|
| **Alliance Player** (view‑only member) | [Roles Explained](roles/overview.md) → [Reading the Dashboard](getting-started/dashboard-tour.md) |
| **Co‑Leader** (helps track and enter data) | [Roles Explained](roles/overview.md) → [Finding Your Way Around](getting-started/navigating.md) |
| **Alliance Leader** (runs one alliance) | [Roles Explained](roles/overview.md) → *My Alliance guide (coming in a later section)* |
| **King** (runs one kingdom) | [Roles Explained](roles/overview.md) → [Safety Rules You'll Run Into](roles/protection-rules.md) |
| **Supreme Admin** (runs the whole platform) | [Roles Explained](roles/overview.md) → [Safety Rules You'll Run Into](roles/protection-rules.md) |
