# Henod Processor

**Henod Processor** is a platform-managed processing option. Users do not add or manage any service details for it.

## Using it

Open Imports or KvK, choose **Free**, then select **Henod Processor**. Upload the screenshot and review the detected rows before applying them.

## Availability

Henod remains visible in the Free category and is selectable only when it is available. If it is unavailable, the picker gives a safe explanation and prevents a new import from selecting it.

Choose another available processor if Henod is unavailable. Administrators can use **Processing Services** to re-check its status.
