# Import History & Change Tracking

The imports page is not just an upload screen. It is also the history view for what was processed, reviewed, applied, rolled back, or restored.

## What import history shows

The history table shows items such as:

- file name
- source
- event
- date
- status
- row count

That makes it the quickest place to answer, "Which import was this?"

## Review page history details

When you open an import review page, you can also inspect:

- the saved result rows
- related record batches
- raw processing data

This is useful when you want both the business view and the evidence behind it.

## Why change tracking matters

Accepted rows leave a change trail that helps rollback work safely later.

That is especially important for:

- created players
- overwritten results
- field updates such as power
