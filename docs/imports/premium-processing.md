# Premium Processing

**Premium Processing** is a platform-managed screenshot-processing option. Unlike bring-your-own-key processors, it needs no personal key from the user.

## When you can use it

Premium Processing appears as selectable only when:

1. an administrator has enabled the service;
2. your current kingdom or alliance includes Premium Processing; and
3. the service is currently healthy.

If it is temporarily unavailable, the app gives a safe status message and offers another processor. It never reveals sensitive service details in the normal user interface.

## How to use it

1. Open **Imports**.
2. Choose **Premium**, then select **Premium Processing**.
3. Upload or paste a screenshot.
4. Review detected rows before accepting them.

All processor output is review-first: detected rows are drafts until you accept them.
