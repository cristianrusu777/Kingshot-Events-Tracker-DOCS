# Login & Account Problems

Use the short path that matches what is stopping you from getting in.

## 1. Wrong password too many times

1. Stop retrying.
2. Wait for the cooldown to pass. The login system is rate-limited, and repeated failures only make it worse.
3. If you still do not know the password, use [Reset a Forgotten Password](../getting-started/forgot-password.md).

## 2. I forgot my password

1. Open [Reset a Forgotten Password](../getting-started/forgot-password.md).
2. Submit the reset request.
3. Contact your King or Supreme Admin if the reset is urgent.

## 3. My account is still pending

1. Re-read [Request an Account](../getting-started/registering.md).
2. Wait for a senior role to review the request.
3. If it has been pending for too long, contact your Alliance Leader, King, or Supreme Admin.

## 4. My request was rejected or needs more information

1. Review [Request an Account](../getting-started/registering.md).
2. If more information is needed, provide it through the process your admins use.
3. If it was rejected and you think that is wrong, contact the approving admin directly.

## 5. My account exists but still cannot get in

1. Make sure you are using the correct username.
2. Check whether the issue is really access after login instead of login itself. If so, read ["You don't have access"](../getting-started/access-denied.md).
3. Ask an admin to review the account in the user-management flow. The relevant admin guide starts at [Create a User](../how-to/create-user.md) and [Edit a User & Reset Their Password](../how-to/edit-user.md).
