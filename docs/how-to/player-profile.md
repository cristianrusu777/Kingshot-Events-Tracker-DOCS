# Read a Player Profile

The player profile is the deeper view of one player. It combines the current profile, sync state, nickname history, attributes, graphs, and recent records in one place.

> Screenshot placeholder: player-profile.png

## Open the profile

Open **Players**, then select the player's name or profile action.

## Main sections on the page

The profile is organized into these sections:

- **Profile Foundation**
- **Sync State**
- **Nickname History**
- **Player Attributes**
- **Graphs**
- **Resolved Records**
- **Events**

## Profile Foundation

This is the main identity area. Depending on your role, you may be able to edit:

- name
- Kingshot profile ID
- external profile URL
- avatar URL
- power
- level
- game rank
- TC level
- alliance
- alliance role
- alliance status
- notes

If you change the alliance from one current alliance to another, the page asks you to confirm the transfer so the membership history is recorded correctly.

## Sync State

This section tells you whether the player is:

- API linked
- local only
- recently synced

It also shows the current status labels the player is carrying.

## Nickname History

Use this area when imports, manual notes, or old event records use a former player name.

You can:

- add a nickname
- make an older nickname the current one
- remove a non-current nickname

## Player Attributes

This section shows the player's current status and any extra attributes from analytics or manual decisions. If your role allows it, you can also add manual attributes and save or clear a manual status override here.

## Graphs and records

The lower parts of the page help you answer questions like:

- how often this player appears
- how their score changes over time
- what name imported rows used
- which events they have been part of
