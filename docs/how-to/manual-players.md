# Add Players During Manual Entry

Sometimes manual result entry shows the right event and date, but one or more players are missing from the loaded alliance list. That is when **Manual players** helps.

## When to use it

Use manual player management when:

- a player is new and not yet in the tracker
- you need to fix the roster before saving a manual batch
- you want to avoid creating the result first and the player later

## Simple workflow

1. Open **Manual Input**.
2. Select **Manual players**.
3. Add or correct the player.
4. Return to the manual result page.
5. Reload the context if needed.

## Why this matters

Manual input loads players by alliance context. If the player does not exist in that context yet, they cannot be entered cleanly into the batch.
