# Edit an Event Template

Edit a template when you want to change how future instances, imports, and analytics for that event behave.

## What you can change

On the template detail and settings pages, you can adjust things like:

- name, icon, badge label, and colors
- descriptions and instructions
- score labels and limits
- duration and stage behavior
- membership and eligibility policies
- dashboard and analytics visibility

## Where to edit

You usually work in two places:

- the template detail page for the main event identity
- the event settings page for the deeper behavior and policy controls

See [Configure Event Settings](event-settings.md) for the settings surface itself.

## Default templates vs custom templates

You can edit both default and custom templates, but there is one big difference:

- default templates are protected from deletion
- custom templates can be deleted when policy allows it

If you need a local version of a default event, use the **Customize** action to create a scoped copy.

## Important warning

Template changes are meant to shape future work. They should not be used with the expectation that old event history will be rewritten.

For example:

- changing a score label helps future clarity
- changing import behavior helps future imports
- old saved rows are still old saved rows
