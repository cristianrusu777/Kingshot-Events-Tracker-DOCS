# Platform Console

The **Platform Console** is the separate live activity page for the whole platform. It is not limited to image processors.

Use it to monitor recent audit activity for:

- authentication
- imports and review actions
- settings changes
- subscription and grant actions
- admin operations
- player, event, and alliance data changes

The page polls recent scoped audit records and presents them as a readable feed with search, entity-type filtering, success/failure filtering, and copy support.

## Access

- **Supreme Admin** sees the global platform view
- **Kingdom-scoped users** see their kingdom activity
- **Alliance-scoped users** see their alliance activity
- users without log permissions do not see the page

This page complements the processor-focused [Processing Console](../imports/processing-console.md).
