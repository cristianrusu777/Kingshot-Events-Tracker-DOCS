# Subscriptions & Usage Manager

This guide is for `Supreme Admin` users only.

The **Subscriptions & Usage** page is the platform-wide dashboard for plans, live usage, direct assignments, overrides, and suspension actions.

> Screenshot placeholder: `subs-admin-overview.png` (S61)

## What this page shows

At the top, the page shows platform summary cards such as:

- total kingdoms
- total alliances
- total plans
- suspended subjects
- warning and critical counts

Below that, the page gives you:

- search by kingdom or alliance name
- filters by status and plan
- a toggle to open the plans area
- separate usage tables for kingdoms and alliances

## Reading the usage tables

Each row shows:

- the subject name
- its current plan
- the plan source
- top usage warnings
- overall status

Selecting a row opens its detail area, where you can review:

- per-resource usage bars
- suspension message
- warnings
- cleanup suggestions
- the last calculation time

## What you can do from here

This page is the launch point for three important actions:

- [Assign Plans Directly](assign-plans.md)
- [Suspend & Unsuspend](suspend-subjects.md)
- set quota overrides for a specific resource

It is also the place where you open [Create & Edit Plans](manage-plans.md).

## Good practice

- Use filters first before taking action.
- Open the row detail before suspending or overriding anything.
- Treat direct assignment as an admin shortcut, not as the normal customer-facing request flow.
