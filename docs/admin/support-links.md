# Configure Support Links

This guide is for `Supreme Admin` users only.

Support links are the payment, donation, and contact links shown around the manual subscription flow.

> Screenshot placeholder: `subs-support-links.png` (S70)

## What you can configure

The Support Links page includes fields for:

- PayPal URL
- Revolut URL
- Discord invite URL
- Discord server name
- BTC, ETH, and USDT addresses
- a crypto QR image URL
- a support message
- a Premium Processing note

## Where these links appear

These links support the manual premium flow described in:

- [Request a Subscription](../subscriptions/request-subscription.md)
- [Payment Instructions & Completing a Request](../subscriptions/payment-instructions.md)

They are how the app points people toward payment or support options without processing payments itself.

## Good practice

- keep only current links active
- make the support message short and clear
- use the Premium Processing note only for that specific coming-soon area
- review links after changing payment methods or community spaces
