# Theme Settings

This guide is for `Supreme Admin` users only.

The Theme Settings page controls the platform's shared appearance settings.

> Screenshot placeholder: `admin-theme.png` (S78)

## What the current theme system includes

The current UI is intentionally simple:

- one curated Light mode
- one curated Dark mode
- a motion toggle for subtle animations
- a reset-to-defaults action

The old custom accent colors and extra theme pickers are no longer part of the current UI.

## What you can do here

- review the current visual system
- turn subtle motion on or off
- save appearance settings
- reset appearance settings to defaults

Users still switch between Light and Dark from the normal header toggle. This page controls the shared theme system behind that behavior.

## Good practice

- leave motion enabled unless your team prefers reduced movement
- use reset if the current theme state feels off
- do not expect per-kingdom or per-alliance color sets from the current UI
