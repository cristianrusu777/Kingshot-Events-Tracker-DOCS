# Strongest Governor

Strongest Governor is the default multi-stage template for the individual ranking event with daily objectives and an overall final leaderboard.

## How this default is set up

- event style: multi-stage
- score entry: one score per row
- default stage count: 6
- default duration: 7 days
- participation tracked: yes

## What to import

For this event, you can track either:

- a specific day's stage standings
- the final or overall total standings

That means the best workflow is:

1. create the full event instance
2. import day screenshots as **Stage Data**
3. import final standings as **Total Data**, if your team tracks it

## How this differs from Alliance Brawl

Strongest Governor is still a multi-stage event, but its default import flow is a little less strict than Alliance Brawl. The app does not force the same explicit stage-selection rule for every import.

Even so, you should still choose the right mode so reports stay clean.

## Good practice

- keep stage screenshots and total standings separate
- review conflict rows before accepting if a player already has a saved result for that phase
- use one instance for the full event window instead of splitting each day into separate events

## Protection note

This is a default template, so it is **editable but not deletable**. See [Safety Rules You'll Run Into](../roles/protection-rules.md).
