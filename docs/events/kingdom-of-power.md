# Kingdom of Power

Kingdom of Power is the default kingdom-wide multi-stage template for the main KvK preparation and competition cycle.

## How this default is set up

- event style: multi-stage
- score entry: one score per row
- default stage count: 5
- default duration: 6 days
- participation tracked: yes

This default is usually used at **kingdom** scope rather than alliance scope.

## What is special about this event

Unlike most alliance events, Kingdom of Power often needs:

- kingdom-wide standings
- phase-by-phase tracking
- separate before/after power capture through KvK Prep

The normal event results and the KvK Prep tool work together, but they are not the same page.

## What to import

Use **Kingdom event results** when you are importing kingdom-wide screenshots for this event.

You can use:

- **Stage Data** for a specific phase
- **Total Data** for overall standings

For this default, the app does not force stage selection in the same strict way as Alliance Brawl, but you should still choose the correct mode yourself.

## KvK Prep is separate

Before/after power snapshots belong in the dedicated KvK Prep workflow, not in normal result rows. See [Run a KvK Prep Session](../how-to/kvk-prep.md).

## Good practice

- create one instance for the full KvK event window
- keep normal event-result imports separate from before/after power snapshots
- use kingdom-result scope when the screenshot is kingdom-wide

## Protection note

This is a default template, so it is **editable but not deletable**. See [Safety Rules You'll Run Into](../roles/protection-rules.md).
