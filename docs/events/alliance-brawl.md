# Alliance Brawl

Alliance Brawl is the default multi-stage template for the long brawl event with daily stages and one overall total view.

> Screenshot placeholder: import-stage-picker.png

## How this default is set up

- event style: multi-stage
- score entry: one score per row
- default stage count: 6
- default duration: 7 days
- participation tracked: yes

This event is stage-based, so the stage choice on import matters a lot.

## The important import rule

When you upload results for Alliance Brawl, you must choose one of these on purpose:

- **Stage Data** and the matching day number
- **Total Data** for the overall standings

For this default, stage selection is not optional or automatic.

That means:

- if your screenshot shows Day 1 results, import it as **Stage Data** and choose **Stage 1**
- if your screenshot shows the overall event standings, import it as **Total Data**

Do not upload a day screenshot while leaving the stage unclear and expect the tracker to guess.

## Why the stage choice matters

Alliance Brawl usually needs two different views:

- per-day stage results
- an overall total or final standings view

Choosing the wrong mode mixes those two ideas and makes reports harder to trust.

## Suggested workflow

1. Create the instance with the full date range.
2. Check the stage dates if your team wants them filled in.
3. Import each day's screenshot as **Stage Data**.
4. Import the overall standings separately as **Total Data**, if you track it.

## Protection note

This is a default template, so it is **editable but not deletable**. See [Safety Rules You'll Run Into](../roles/protection-rules.md).
