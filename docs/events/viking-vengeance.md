# Viking Vengeance

Viking Vengeance is the default template for the alliance defense event where players earn **Points** by killing troops and helping with the waves.

## How this default is set up

- event style: single-session
- score entry: one score per player
- score label: **Points**
- score required: yes
- participation tracked: yes

Like Bear Trap, this is a simple one-session score event.

## What to import

Upload the personal points or ranking screenshot after the event finishes.

There is:

- no stage selection
- no total-vs-stage choice
- no cumulative daily snapshot model

## When this template works well

Use it when you want a clean record of:

- who joined the defense
- each player's final points
- who met your reward or attendance rules

## Good practice

- create one instance per event date
- review low-confidence names before accepting all
- use manual input only for cleanup, not as the main path when you already have a clear ranking screenshot

## Protection note

This is a default template, so it is **editable but not deletable**. See [Safety Rules You'll Run Into](../roles/protection-rules.md).
