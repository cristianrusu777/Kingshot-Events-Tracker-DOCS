# Stages, positions, and resources

A **stage** represents a scheduled KvK day or event part. A cycle can use up to five configured stage days. Every stage can have its own date, UTC start time, availability, resources, eligibility settings, and schedule.

Positions are schedule columns. Administrators can activate positions, set their order, duration, slot count, and capacity. Positions can inherit stage timing or use their own timing. A stage with no configured positions cannot build a position schedule.

The Kingdom can make a resource public, required, eligible for ranking, or subject to a minimum. Icons make resource categories easier to recognise. True Gold is an amount; speedups are durations.

Only an in-scope King, Minister of Justice, or Supreme Admin can configure these controls. Players see only active public fields.
