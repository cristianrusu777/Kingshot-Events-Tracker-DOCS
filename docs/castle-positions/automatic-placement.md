# Automatic placement suggestions

The planner can calculate suggestions for a stage. This is a **preview**, not an automatic final decision or publication.

Suggestions consider eligible candidates, rank/score, locked assignments, capacity, and availability. Results identify an exact preferred time, alternative choice, nearby time, or any available time. A nearby result is not the same as the requested hour.

When a slot is full, a candidate may be suggested elsewhere or left unplaced with a reason and recommended action. Administrators must review, change if needed, and save the result before finalizing and publishing.
