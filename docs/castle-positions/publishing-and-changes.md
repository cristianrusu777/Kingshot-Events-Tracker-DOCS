# Publishing and schedule changes

1. **Draft / saved:** editing continues; players cannot rely on it.
2. **Finalized:** validated and locked for review, but still not player-visible.
3. **Published:** player-visible and notifications are attempted for affected opted-in applicants.
4. **Reopened:** editing resumes; players retain the last published version until the revised version is published.

Each publication creates the next published version. The player overview and appointment history show effective schedule information. Reopening cannot retract an email already sent, so communicate urgent changes directly as well.
