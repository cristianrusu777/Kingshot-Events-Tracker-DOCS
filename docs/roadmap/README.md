# Future Roadmap

> **Status: Planned / Future - not implemented yet.**

This section collects product ideas for possible future versions of Kingshot Event Tracker. It is planning material, not a promise, release announcement, entitlement, or description of the current application.

## Roadmap labels

| Label | Intended future audience |
|---|---|
| **Free** | Baseline improvements intended to make core tracking safer and easier for everyone. |
| **Premium Alliance** | Deeper player and alliance insight for leaders and co-leaders. |
| **Premium Kingdom** | Cross-alliance oversight, comparison, governance, and kingdom reporting. |
| **Premium Both** | A premium capability planned for both alliance and kingdom plans, with scope-appropriate data. |
| **Future Experimental** | Research ideas that need validation, strong safeguards, and possibly an opt-in pilot. |

Final packaging may change after product research. A tier label here does not mean the feature is currently available.

## Product principles

- Important recommendations must be explainable and reversible.
- Imports remain drafts until a person reviews and accepts them.
- Prediction is decision support, never an undisclosed automatic judgment.
- Premium packaging must not weaken basic data safety or privacy.
- Mobile and capture tools must not automate gameplay or bypass game protections.
