# Data Quality & Review Improvements

> **Planned / Future · Not implemented yet · Free foundation with Premium Both dashboarding**

## Planned Free safeguards

- clearer OCR confidence reasons
- duplicate row and duplicate result detection
- normalized name and numeric-unit previews
- source image/frame trace for every detected value
- conflict comparison showing existing versus proposed value
- “why this row needs review” explanations
- same-alliance membership guards
- stale roster and missing event-context warnings
- bulk accept grouped by risk, never one undifferentiated action
- correction history and undo guidance

## Planned Premium Both capabilities

- data-quality dashboard by source, event, alliance, and time window
- recurring OCR-error pattern discovery
- import/provider quality comparison
- review workload trends
- manual-override analysis
- completeness scoring with drill-down
- configurable quality thresholds for reports and recommendations

## Quality score components

A future quality score may summarize match coverage, confidence coverage, conflict rate, missing context, unresolved age, and manual correction rate. It must never hide severe issues behind a healthy average.

## Acceptance rule

Screenshot, spreadsheet, and future video inputs remain drafts. No player or result data is saved until an authorized user reviews and accepts the relevant rows.
